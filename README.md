# AR-Mine
AR Mine - Minecraft in augmented reality

## О игре
### Игра представляет собой майнкрафт в дополненной реальности.
Игра делается на игровом движке Unity 2020.3.26f1

В игре используется **AR Core**

**AR Core - Набор для разработки программного обеспечения, разработанный Google, который позволяет создавать приложения дополненной реальности.**

В игре перенесена основная механика создание построек
